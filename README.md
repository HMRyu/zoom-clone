## Zoom Clone Coding

실시간 회의를 할 수 있는 Zoom 서비스를 클론 코딩해보며 이해도를 높이려고 하였습니다.

## Tech Stack

- Next.js
- TypeScript
- Clerk
- stream.io
- Tailwind CSS
- shadcn

## Quick Start

### Clone
```
git clone https://github.com/HMRyu/zoom-clone.git
```

### Installation
```
npm install
```

### Run
```
npm run dev
```
